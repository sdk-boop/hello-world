§ 01

原文内容的逻辑分析

这段论文内容的逻辑链条非常清晰，本质上是一个 "现象 → 建模 → 参数标定 → 模量推导" 的标准动本构研究范式。

一、整体逻辑骨架

非线性现象观察 → 双曲线模型假设(5.26) → 试验拟合得 a、b (表5.11) ↓ 模量定义 Ed = σd/εe ↓ 反演得 Ed-εe 关系(5.27) ↓ 画出 Ed-εe 曲线(图5.56、5.57)

二、关键数学链条

从 (5.26) 到 (5.27) 不是凭空写出来的，而是有严格推导。由定义 $E_d = \sigma_d / \varepsilon_e$，代入 (5.26)：

$$E_d = \frac{\sigma_d}{\varepsilon_e} = \frac{1}{a + b\varepsilon_e}$$

这就是 (5.27)。重点在于它的倒数形式：

$$\frac{1}{E_d} = a + b\varepsilon_e$$

这是一条直线！这就是文中那句"对上述双曲线模型 线性回归"的真正含义——把 $1/E_d$ 对 $\varepsilon_e$ 作图，截距即 $a$，斜率即 $b$。这是 Hardin-Drnevich 原始论文里就用的经典手法。

三、试验设计的逻辑（表 5.11）

表 5.11 的因子设计是 三因素正交式 铺开的：

  • 冷却温度：-5°C vs -10°C（冻结深度影响）
  • 冻融次数：0、1、6（损伤累积影响）
  • 围压：20、50、80 kPa（应力状态影响）

这种铺法是为了让 $a$、$b$ 能写成这三个因素的函数，从而把模型推广到任意工况。对你尤其有参考价值：你的红黏土如果要考虑围压相关性，标定后也需要做类似的"参数-围压回归"才能用于 FDEM 输入。

四、参数变化趋势的物理含义

粗看表中数据可读出：

  • 围压增大时，$a$ 总体趋于增大 → 初始模量随围压增加（符合常理，$E_{d,\max} = 1/a$）
  • 冻融次数增多时，$a$ 也明显增大 → 冻融损伤后土体软化，初始模量下降
  • -10°C 比 -5°C 的 $a$ 普遍更大 → 冻结越深，损伤越严重
⚠ 数据排版错误警告

表中有两个数据明显异常，几乎肯定是原文 OCR 或排版错误，引用时务必核对原始论文：

  • -10°C / 冻融 1 次 / 80kPa：$b = 650\text{E-}4$（其它都是 $10^{-4}$ 量级的个位数，应为 $6.50\text{E-}4$）
  • -10°C / 冻融 6 次 / 20kPa：$b = 6.1\text{E-}6$（应为 $6.1\text{E-}4$）

五、与你研究的对照

这篇用的是 Hardin-Drnevich 基础双曲线形式（只有 $a$、$b$ 两参数，仅描述 $\sigma_d$-$\varepsilon_e$ 骨干曲线）。你的 H-D 耦合模型 多出 $F_\varepsilon$、$m$、$n$、$p$ 等参数，是为了同时刻画 骨干曲线 + 阻尼比演化 + 围压依赖性——这是从"静态曲线拟合"升级到"动力本构"的关键差异。所以你看这篇可以借鉴它的 试验工况设计逻辑 和 线性化标定流程，但参数体系不能直接套用。

§ 02

线性拟合 vs 线性回归（严格定义）

这两个词在工程文献里经常被 混用，但严格说它们的含义有微妙差别。更重要的是——在原文里它们其实 不是两个步骤，而是 同一件事。

一、严格定义上的区别

概念辨析

线性拟合 (Linear Fitting) — 指"用一条直线去逼近一堆散点"这件事本身。它强调的是 几何动作：找一条 $y = kx + c$ 让它尽量靠近所有数据点。

线性回归 (Linear Regression) — 指用 统计学方法（通常是最小二乘法 OLS）求出这条直线的参数 $k$ 和 $c$，并附带评估拟合质量（$R^2$、残差、置信区间等）。它强调的是 数学求解过程。

打个比方：

  • "线性拟合" ≈ "我要画一条直线穿过这些点"（目标）
  • "线性回归" ≈ "我用最小二乘法算出这条直线的斜率和截距"（手段）

所以严格讲，线性回归是实现线性拟合的一种具体方法。日常工程文献里两者基本等价。

二、为什么原文里看起来像"两个步骤"

"采用双曲线模型对动应力-应变试验数据进行 拟合，即符合式(5.26)"

"通过对上述双曲线模型 线性回归 后，就可以得到... $E_d^{-1}$~$\varepsilon_e$ 关系"

它们指的其实是 同一个数学操作，但 作用的对象不同：

第一句的"拟合"——指目标

作者是在说："我假设 $\sigma_d$ 和 $\varepsilon_e$ 的关系符合双曲线 (5.26)，要去拟合它"。这是 陈述建模目标，不是描述具体操作。

第二句的"线性回归"——指手段

但是 (5.26) 本身是 非线性的：

$$\sigma_d = \frac{\varepsilon_e}{a + b\varepsilon_e}$$

直接对这个非线性式求 $a$、$b$ 要用 非线性最小二乘（迭代算法，麻烦），所以作者用了一个经典技巧——变量代换线性化：

两边取倒数 → $\dfrac{\varepsilon_e}{\sigma_d} = a + b\varepsilon_e$

或者用模量形式 → $\dfrac{1}{E_d} = a + b\varepsilon_e$

令 $y = 1/E_d$，$x = \varepsilon_e$，原本的双曲线就变成了一条直线 $y = a + bx$！这时候再做最小二乘法（线性回归），就能 一步求出 $a$（截距）和 $b$（斜率），不需要迭代。

三、为什么非要"线性化"？历史原因

这个套路是 1972 年 Hardin & Drnevich 原论文里就定下来的，原因有三：

  1. 计算简单：那个年代没有现成的非线性求解器，线性回归用计算尺、手摇计算器都能算。
  2. 结果直观：在 $1/E_d$~$\varepsilon_e$ 图上点应该排成直线，一眼就能看出模型是否适用——如果点散乱不成线，说明双曲线假设不对。
  3. 物理意义清晰：
    • 截距 $a = 1/E_{d,\max}$ → 小应变下的最大动模量
    • $1/b$ → 极限动应力 $\sigma_{d,\text{ult}}$
★ 关键洞察

线性化不只是计算技巧，更是一种模型验证手段。这也是为什么直到今天 Hardin-Drnevich 标定还坚持用这种方法。

§ 03

高中生版解释

一、最直白的解释

假设你想研究 "喝奶茶的杯数" 和 "长胖的斤数" 有没有关系。你记录了 10 个同学的数据，画在坐标纸上：

  • 横轴 $x$：每周喝几杯奶茶
  • 纵轴 $y$：一个月胖了几斤

你会发现这 10 个点 乱糟糟散在那里，但 大致有个上升的趋势——喝得越多胖得越多。这时候你想：能不能画一条直线，把这个趋势表示出来？

什么是"线性拟合"

你拿起尺子，在这堆点中间画一条直线，让它 尽量穿过点群中央，让大部分点都离这条线很近。这件事就叫 线性拟合——用一条 直 线去 贴合 一堆数据点。

什么是"线性回归"

但是用尺子凭眼睛画，每个人画的都不一样。数学家就发明了一个公式，输入你的数据点，自动算出"最佳直线"的斜率和截距。这个用公式自动算最佳直线的方法，就叫 线性回归。

✓ 一句话区分
  • 线性拟合 = 我想干的事（画直线贴近数据）
  • 线性回归 = 我用的工具（数学公式帮我算出来）

就像"想吃米饭"和"用电饭煲煮饭"的关系。一个是目标，一个是手段。

二、那论文里为什么搞得这么复杂？

因为论文里的数据点 不是直线趋势，是弯曲的！

横轴是"应变"（土被压变形了多少），纵轴是"应力"（土受了多大的力）。画出来的点是一条 弯弯的曲线：

应力 ↑ | _____ ← 越往后越平 | / | / | / |/_____________→ 应变

这种弯曲形状叫 "双曲线"。问题来了：我们刚学的工具叫 "线性回归"——它只会处理 直线，不会处理曲线！

三、论文里的聪明做法：把曲线"掰直"

数学家有个绝招：通过换变量，把弯曲的关系变成直线的关系。

举个简化的例子，假设原来的关系是 $y = \dfrac{1}{a + bx}$（一条曲线），两边取倒数：

$$\frac{1}{y} = a + bx$$

看！ 如果我们令 $Y = 1/y$，那么 $Y = a + bx$，这就是一条 直线 了！斜率是 $b$，截距是 $a$。

  • 原来画 $y$ 和 $x$ 的关系 → 弯曲
  • 改画 $1/y$ 和 $x$ 的关系 → 变直了

然后我就可以用线性回归这个工具，求出斜率 $b$ 和截距 $a$。

四、完整流程串起来

1. 土的"应力-应变"数据是弯的（双曲线） ↓ 2. 我假设它符合双曲线公式 (5.26) ← 这步叫"拟合"（确定目标形状） ↓ 3. 我用"取倒数"的技巧，把曲线掰成直线 ↓ 4. 对掰直后的数据用最小二乘公式 ← 这步叫"线性回归" ↓ 5. 算出 a 和 b 两个参数 ↓ 6. 把 a、b 代回原公式，弯曲关系就完全确定了
★ 一句话总结

"拟合"是说"我要把曲线方程对上数据"这件事；"线性回归"是说"我具体怎么算出方程里的未知数"的方法。论文里把曲线先掰直，再用线性回归算参数，是因为 算直线比算曲线简单太多了。

这其实是一个非常聪明的偷懒技巧——不会处理曲线？那就想办法把它变成直线再处理。

§ 04

流程与原文逐句对照

把刚才那个"完整流程"的 6 个步骤，逐条对应回原文。这样就能明白论文里每句话、每个公式分别对应流程里的哪一步。

步骤 1：土的"应力-应变"数据是弯的

"线具有明显的 非线性 特征，在周期荷载作用下..."

"非线性"就是"不是直线"的意思，也就是数据画出来是 弯的。这是作者观察试验数据后得出的现象。

步骤 2：假设符合双曲线公式 (5.26)——"拟合"

"采用 双曲线模型 对动应力-应变试验数据进行 拟合，即符合式(5.26)。"

$$\sigma_d = \frac{\varepsilon_e}{a + b \cdot \varepsilon_e} \quad (5.26)$$

这就是确定"曲线长什么样"——作者说："我认为土的应力应变关系符合这条双曲线，里面有两个未知数 $a$ 和 $b$ 等着我去算。"

步骤 3：用"取倒数"的技巧把曲线掰直

原文里没明说这一步，但它 藏在公式 (5.27) 里。来看作者是怎么操作的：

原式 (5.26)：$\sigma_d = \dfrac{\varepsilon_e}{a + b\varepsilon_e}$，两边同时除以 $\varepsilon_e$：

$$\frac{\sigma_d}{\varepsilon_e} = \frac{1}{a + b\varepsilon_e}$$

而动模量的定义就是 $E_d = \dfrac{\sigma_d}{\varepsilon_e}$（动应力除以动应变），所以：

$$E_d = \frac{1}{a + b\varepsilon_e} \quad (5.27)$$

这就是原文公式 (5.27) 的由来！ 它不是凭空冒出来的，而是 (5.26) 变形得到的。再取一次倒数（关键的一步！）：

$$\frac{1}{E_d} = a + b\varepsilon_e$$

到这里曲线就被"掰直"了——左边 $1/E_d$ 是 $y$，右边 $\varepsilon_e$ 是 $x$，关系变成 $y = a + bx$，一条直线！

步骤 4：对掰直后的数据用线性回归

"通过对上述双曲线模型 线性回归 后，就可以得到水泥改良土和石灰改良土试样的 $E_d^{-1}$~$\varepsilon_e$ 关系..."

注意作者写的是 "$E_d^{-1}$~$\varepsilon_e$ 关系"——这里的 $E_d^{-1}$ 就是 $1/E_d$！这恰好印证了步骤 3——作者就是在 $1/E_d$ 和 $\varepsilon_e$ 这个掰直后的平面上做线性回归的。

步骤 5：算出 a 和 b

对应 表 5.11 整张表。表里每一行就是 一次线性回归的结果——换一组工况（温度/冻融次数/围压），就重新做一次"掰直+回归"，得到一对新的 $a$、$b$。整张表 18 行，就代表作者做了 18 次线性回归。

步骤 6：代回原公式，画出完整关系曲线

"根据公式 (5.27)，可以得到 动回弹模量与动应变的关系曲线，图 5.56 和图 5.57 是..."

把表 5.11 里算出来的 $a$、$b$ 代回 (5.27)，就能画出 $E_d$ 随 $\varepsilon_e$ 变化的 完整曲线。到这里整个流程就闭环了：数据 → 假设公式 → 掰直 → 回归 → 求参数 → 还原曲线。

💡 小提醒

原文为什么读起来跳跃、有点难懂？因为作者把 步骤 3（掰直的过程）省略了。他直接从 (5.26) 跳到 (5.27)，又突然说"线性回归"——对没学过这个套路的人来说会觉得莫名其妙。

真相：作者心里走的是 (5.26) → 取倒数变形 → (5.27) → 再取倒数变直线 → 线性回归。但他笔下只写了首尾，中间靠读者自己补。这是 工程论文的通病——默认读者懂行。

§ 05

什么是归一化处理

一、最直白的解释

核心定义

归一化 (Normalization) = 把数据"统一标准"，让不同的东西能放在一起比较。

"归一"两个字字面意思就是"归到 1"——把一堆数据压缩到 0 到 1 之间，或者以某个基准为 1 来衡量其他值。

二、生活中的归一化例子

例子 1：考试成绩

你这次月考语文考了 85 分（满分 100），数学考了 120 分（满分 150）。请问你哪科考得更好？

直接比 85 和 120 没意义，因为 满分不一样。怎么办？把它们都换算成"得分率"：

  • 语文：85 / 100 = 0.85
  • 数学：120 / 150 = 0.80

哦，原来语文考得更好。这个"除以满分"的操作，就是归一化——把不同尺度的东西，统一到 0~1 这个范围里来比。

例子 2：班级身高

你们班最矮的同学 150 cm，最高的 190 cm。想把每个人的身高表示成"在班里的相对水平"，可以这样算：

$$\text{归一化身高} = \frac{\text{你的身高} - 150}{190 - 150}$$
  • 最矮的人：(150-150)/40 = 0
  • 最高的人：(190-150)/40 = 1
  • 170 cm 的人：(170-150)/40 = 0.5 ← 刚好在中间

这样一来，每个人的身高都变成了 0~1 之间的一个数，一眼就能看出在班里的位置。

三、归一化的三种常见做法

方法 公式 典型用途
1. 最大值归一化 $x' = x / x_{\max}$ 成绩除以满分
2. 最大-最小归一化 $x' = (x - x_{\min}) / (x_{\max} - x_{\min})$ 身高拉伸到 0~1
3. 除以基准值 $x' = x / x_{\text{基准}}$ 物价指数（以 2000 年为 1）

四、归一化的三大作用

作用 1：让不同量纲的东西能比较

身高用 cm，体重用 kg，年龄用岁——单位都不一样，没法直接比谁"更突出"。归一化后都变成纯数字，就能比了。

作用 2：消除"大小"的干扰，看出本质规律

这是科研里最重要的用途。假设你测三块土样的应力-应变曲线，三个样的"最大应力"分别是 100、200、300 kPa。直接画三条曲线，它们高低差很大：

应力 ↑ | ___ 样3 (max=300) | ___ | __ 样2 (max=200) | _ 样1 (max=100) |_______________→ 应变

但如果 把每条曲线的应力都除以它自己的最大值（归一化），三条曲线就会 叠在一起：

σ/σmax ↑ 1.0 | ___ 三条曲线重合！ | / |/_______________→ 应变

这说明什么？ 说明三块土样虽然强度不同，但它们的 变形规律是一样的。归一化帮你 剥掉了"大小"的外衣，露出了"形状"的本质。

作用 3：让计算更稳定

在做计算（尤其是机器学习、有限元）时，如果有的数是 0.001，有的数是 1000000，计算机算起来容易出错。归一化把所有数压到差不多大小，计算就稳了。

五、对应到你的研究里

例子 A：Hardin-Drnevich 模型的归一化

经典的 H-D 公式经常写成：

$$\frac{G}{G_{\max}} = \frac{1}{1 + \varepsilon/\varepsilon_r}$$

注意左边 $G/G_{\max}$——动剪切模量除以 最大动剪切模量，这就是归一化！

为什么这么做？ 因为不同土样的 $G_{\max}$ 千差万别（从几十到几百 MPa 都有），但 $G/G_{\max}$ 这个 归一化模量 的衰减规律对很多土都差不多。这样画出来的 $G/G_{\max}$~$\varepsilon$ 曲线就成了一条 "通用曲线"。

例子 B：围压归一化

很多论文写参数时会用：

$$\frac{\sigma_d}{p_a} \quad \text{或} \quad \frac{\sigma_3}{p_a}$$

这里 $p_a$ 是 大气压（约 101 kPa）。把应力除以大气压，就变成了一个"多少个大气压"的纯数字。这样不管你用 kPa 还是 MPa，公式都通用。

例子 C：你研究中可能用到的

你做红黏土动三轴试验，如果想把 不同围压下的曲线放在一起对比，归一化就特别有用：

  • $\sigma_d / \sigma_{d,\max}$：归一化动应力
  • $\varepsilon / \varepsilon_f$：归一化应变（$\varepsilon_f$ 是破坏应变）
  • $E_d / E_{d,\max}$：归一化模量

这样 剥离围压影响后，能更清楚地看到红黏土本身的变形规律。

★ 一句话总结

归一化 = 把数据除以某个基准，让不同尺度的东西能放在同一把"尺子"上比较，露出本质规律，去掉表面差异。

打个比方：归一化就像 把不同身高的小朋友都拉到 1 米来比胖瘦——身高差异这个"干扰因素"消除后，你才能看清谁真的胖、谁真的瘦。在研究里，我们经常需要"消除某个因素，看清另一个因素"，归一化就是干这事的。
